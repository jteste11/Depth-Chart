## Installation

Follow these steps to set up the project:

1. **Clone the repository:**
    ```sh
    git clone https://github.com/jteste11/DepthChart.git
    ```
2. **Open the project in Visual Studio:**
    - Open Visual Studio.
    - Select `File` > `Open` > `Project/Solution`.
    - Navigate to the cloned repository and select the `.sln` file.

3. **Restore NuGet packages:**
    - Right-click on the solution in the Solution Explorer.
    - Select `Restore NuGet Packages`.

4. **Build the project:**
    - Press `Ctrl+Shift+B` or select `Build` > `Build Solution`.

## Usage

Instructions on how to use the BaseballDepthChart application:

1. **Run the application:**
    - Press `F5` or select `Debug` > `Start Debugging`.

2. **Using the application:**
    - Once the application is running, you can search for baseball team depth charts by selecting the team name in the search bar.
	- Save spreadsheet by clicking the Export button
	- Check today's games by selecting the Today's games button
	- View pitching matchups by clicking the game you want to see
	
## Features

- Search for baseball team depth charts.
- Display detailed information about team players and positions.
- Export roster details to spreadsheet
- View today's games and pitching matchups
- User-friendly interface for easy navigation.
